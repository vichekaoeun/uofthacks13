# NowAndThen - UofTHacks13

**Discover the identity of places through the memories people leave behind.**

NowAndThen is a location-based map where nearby comments appear as you move. Add your own memory at a spot, explore what others left around you, and watch places build their identity over time.

## Features
- See **nearby memories** pinned to locations on the map
- **Add a comment** from your current location
- **Follow mode + recenter** for a smooth walking experience
- **Adjust discovery radius** (25m to 1km)
- Optional: **trail playback** for users with multiple posts (if enabled)

## Tech Stack
- **Mobile:** React Native (Expo), `react-native-maps`
- **Backend:** Node.js + Express
- **Database:** MongoDB
